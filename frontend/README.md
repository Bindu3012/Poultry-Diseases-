# 🐔 Poultry Disease Classification Web Application

## 📌 Project Overview

This project uses Transfer Learning (MobileNetV2) to classify poultry diseases into:

- Salmonella
- Newcastle Disease
- Coccidiosis
- Healthy

Model Accuracy: **94%**

---

## 🧠 Technologies Used

- Python 3.10
- TensorFlow 2.10
- Flask
- HTML, CSS, JavaScript
